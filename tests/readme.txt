For each test case `.*{i}` with i > 1:
- `readme.txt` describes the purpose of the test.
- `trended.csv` and `simulated.csv` respectively contain reference and new data to compare.
- `param.json` stores the parameters used to run `funnel`.
- `results` directory stores the results generated by `funnel`.
- `temp-plot.html` plots the results generated by `funnel`.
